# Changelog

## CentaurAI distribution

- Rebranded the desktop application as TOKEN MANAGER.
- Added a dedicated horse-shield keyhole icon set for desktop, mobile, installer, and macOS tray targets.
- Redesigned the interface around the shared CentaurAI warm-cream, clay, gold, and tech-dark visual system.
- Removed the preset-provider directory, promotional links, sponsor assets, and upstream update flow.
- Kept custom provider configuration and compatibility with existing CC Switch data, identifiers, and deep links.
- Consolidated secondary features into a client-aware Tools menu.
- Split heavy settings, provider forms, MCP, Skills, Sessions, OpenClaw, and Hermes screens into on-demand bundles.

For upstream history before the CentaurAI distribution, see the original [CC Switch repository](https://github.com/farion1231/cc-switch).
